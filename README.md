# Vending Machine Controller

This repository contains the Vending Machine Controller implemented using Moore and Mealy FSM in Verilog. The project demonstrates the design and functionality of a vending machine controller based on Finite State Machines.

## Features
- Moore and Mealy FSM designs
- Verilog implementations

## Getting Started

This project requires a simulation environment that supports Verilog.
